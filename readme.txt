== Description ==

A simple plugin to append Adally's Accessibility Widget script on your website without having you to modify your frontend code.

== Installation ==

- Download the plugin from your WordPress website
- Install the the plugin
- Click `activate` and our plugin will append a script on your website that will generate the `accessibility widget`

== Frequently Asked Questions ==

#### How to use the plugin?
- *To use our plugin, just install it on your plugin page. Once the plugin is installed, just click `activate` and our plugin will append a script on your website that will generate the `accessibility widget`.*

#### Do we need to install another plugin to make it work?
- *Our plugin will work on it's own. No dependencies from other plugins needed.*

#### Do we need any programming knowledge in order to make the plugin work?
- *You don't have to. Our plugin will do its job for you once it is activated. All you have to do is to sit back and relax!*

#### What is the minimum WordPress version requirement do we need to have in order for this plugin to work?
- *Our plugin should work from WordPress `v1.5.1` up to the latest version which is currently `v5.5.0` by the time we publish this. We will be updating our code once it will start deprecating.*
